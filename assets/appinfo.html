Application Information

This application shows information about the applications stored on your device and optionally about the Android build. Its primary purpose is to let you save this information to the SD card. For the downloaded apps in particular, it lets you keep track of what you installed in case the phone is broken or a hard reset is performed.

Menu Items

Refresh

Refreshes the information.

Save To SD Card

Saves the information to the SD card. The file is named ApplicationInfo.txt and appears in the root directory of the SD card. It is overwritten each time. If you need to save previous versions, you must do it manually.

Copy

Copies the information to the clipboard, so you can paste it into another application, such as email.

Settings

Allows you to change the settings:

  • Show Build Info
  • Show Downloaded Applications
  • Show System Applications

Help

Displays this page.
