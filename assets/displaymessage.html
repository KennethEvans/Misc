Message

This screen displays a message along with its ID, address, time stamp, and body. You can navigate to other messages, delete this one, or fix its time stamp.

Menu Items

Previous

Navigates to the previous message (down in the list).

Next

Navigates to the next message (up in the list).

Delete

Deletes the message and displays the next one.

Fix Time

Brings up a dialog to set the desired change for the time stamp in hours and minutes. The intial value for the change is your time zone offset, which is often the correct value. However, you may not want to use that. Whatever time you enter will be remembered for the next time, whether you cancel or not. If you click OK, the changes will be made or simulated, depending on the current setting for Dry Run.

Set Dry Run

Allows you to set whether fixing the time will be a dry run or a real change to the database. In the case of a dry run, a popup will inform you of the change that would have been made, but no changes are actually made.

Help
